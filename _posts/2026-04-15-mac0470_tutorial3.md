### 4) Installing Linux kernel modules

Ao executar o primeiro comando deste tópico o sistema retornou o seguinte erro:

```bash
root@localhost:~# modinfo simple_mod
modinfo: ERROR: Module simple_mod not found.
```

Esse erro diz que o arquivo simple_mod.ko não foi encontrado, e realmente não estava lá. Para isso tentei transferir o arquivo para a VM do host, Mas também retornou um erro:

```
(LK-DEV) aline@aline:~/lk_dev/iio$ kw ssh --send drivers/misc/simple_mod.ko
bash: line 1: rsync: command not found
rsync: connection unexpectedly closed (0 bytes received so far) [sender]
rsync error: error in rsync protocol data stream (code 12) at io.c(232) [sender=3.4.1]
An error occurred while uploading the file(s). rsync return code: 12
```

Já este erro acontece pois na máquina virtual não tem instalado o rysinc.
