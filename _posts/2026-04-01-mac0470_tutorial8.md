## Compile, Load, and Unload iio_dummy

### Não Consigo Instalar os Módulo dentro da VM

Tentei instalar os módulos dentro da máquina virtual, no entanto ele me retornou o seguinte erro:

```bash
sudo make modules_install
make: *** No rule to make target 'modules_install'.  Stop.
```

Tentei várias soluções, mas nenhuma resolveu o problema. Então tive que utilizar outro método para isso, e construí os módulos manualmente. Alterei o `.config`, reexecutei o comando `kw build` e enviei os arquivos `.ko` para a VM. No entano apenos o arquivo `iio_dummy.ko` não era suficiente, pois ela dependia de outros arquivos, arquivos que enviei um por um para a VM até funcionar.

```bash
#Host
kw build
kw ssh --send drivers/iio/dummy/iio_dummy.ko
kw ssh --send drivers/iio/dummy/iio_dummy_evgen.ko
kw ssh --send drivers/iio/industrialio--sw-device.ko
kw ssh --send drivers/iio/industrialio-configfs.ko

#VM
cp iio_dummy.ko /lib/modules/`uname -r`/kernel/drivers/iio/dummy
cp iio_dummy_evgen.ko /lib/modules/`uname -r`/kernel/drivers/iio/dummy
cp industrialio-sw-device.ko /lib/modules/`uname -r`/kernel/drivers/iio/
cp industrialio-configfs.ko /lib/modules/`uname -r`/kernel/drivers/iio/
```

## Update *read_raw() for Handling the New Channel

### Função iio_dummy_read_raw está Diferente

Procurei pela função `iio_dummy_read_raw`. Mas o `case IIO_CHAN_INFO_RAW` estava muito diferente da do tutorial, e dando uma olhada no bloco do case tinha outra função `__iio_dummy_read_raw`. Esta função parecia mais com a do tutorial e foi neste bloco que fiz as alterações orientados pelo tutorial.

No entanto, quando executei o `kw build` para gerar o arquivo iio_dummy.ko o sistema retornou um erro no `iio_dummy_read_raw`. Observando os outros case da função foi possível observar que nenhum deles estava utilizando o comando `break` e sim o `return`. A seguir como foi instruido pelo tutorial e em sequência as alterações feitas.

```c
// Original
case IIO_MAGN:
    switch(chan->scan_index) {
        case DUMMY_MAGN_X:
            *val = st->buffer_compass[0];
            break;
        case DUMMY_MAGN_Y:
            *val = st->buffer_compass[1];
            break;
        case DUMMY_MAGN_Z:
            *val = st->buffer_compass[2];
            break;
        default:
            *val = 99;
            break;
    }
    ret = IIO_VAL_INT;
    break;

case IIO_CHAN_INFO_SCALE:
	switch (chan->type) {
	case IIO_VOLTAGE:
	    ...
		break;
	case IIO_MAGN:
		// Just add some dummy values
		*val = 0;
		*val2 = 2;
		ret = IIO_VAL_INT_PLUS_MICRO;
		break;
	default:
		break;
	}
```
